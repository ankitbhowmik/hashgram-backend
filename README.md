# hashgram-backend made using Node/ Express JS

steps
1. clone backend :- https://github.com/ankitbhowmik/hashgram-backend.git
2. on terminal run npm install
3. run npm start

4. clone frontend :- https://github.com/ankitbhowmik/hashgram.git
5. run npm start
